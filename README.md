# About
A minimalist desktop experience using Hyprland, focused on productivity and minimizing distractions. 

This repo contains config files for:

* [Hyprland](https://hypr.land/)
* [Hyprlock](https://github.com/hyprwm/hyprlock/)
* [Hyprpaper](https://github.com/hyprwm/hyprpaper)
* [ghostty](https://ghostty.org/)
* [rmpc](https://rmpc.mierak.dev/)
* bash

My neovim config can be found [here](https://github.com/cfsen/kickstart.nvim).

I use [shunpo](https://github.com/cfsen/shunpo) as a minimalist taskbar and application launcher.

# Usage
Clone this repository, and customize the individual configurations to meet your requirements. Alternatively, run `deploy.sh` to use the entire config as is. 

EndeavourOS/Arch compatible package installers can be found in the `install` directory.
